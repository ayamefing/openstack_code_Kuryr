#### kuryr_mitaka_milestone.rst
Mitaka 版本计划的主要工作。

主要有：

* 支持 libnetwork remote API；
* 支持配置认证、连接信息等；
* 虚接口的绑定和解绑定操作；
* 跟 Kolla 的支持；
* 在 VM 里面跑容器的模式支持；
* 支持 Magnum；
* 跟 Neutron 的 API，ID 等的映射；
* CI、打包等。
